# NYU-DataBase-Systems
This is my final project of NYU course "Database Systems"  
  
This is a miniature relational database with order  
Operations supported:  
inputfromfile  
select  
project  
avg  
sumgroup  
avggroup  
countgroup  
join  
sort  
movavg  
movsum  
Btree (construct index)  
Hash (construct index)  
concat
